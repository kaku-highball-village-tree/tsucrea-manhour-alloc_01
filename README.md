# tsucrea-manhour-alloc_01

## 新ローデータ step0002 生成仕様

`新_ローデータ_シート_step0002_YYYY年MM月.tsv` は、次の 2 ファイルが同一実行で検出されたときに生成されます。

- `支給・控除等一覧表_給与_step0001_*.tsv`
- `新_ローデータ_シート_step0001_YYYY年MM月.tsv`

### 入力仕様

#### 1. 給与 step0001 TSV

- 先頭 2 列を `従業員名 -> スタッフコード` の対応として読み込みます。
- ヘッダ行（`従業員名`, `スタッフコード`）は自動除外します。
- 対応表が空の場合はエラーになります。

#### 2. 新ローデータ step0001 TSV

- 各行の 1 列目（氏名）が空欄なら、直前行の氏名を引き継ぎます。
- 氏名をキーに給与 step0001 からスタッフコードを引きます。

### 出力仕様（step0002）

- 出力ファイル名は、入力 step0001 名の `_step0001_` を `_step0002_` に置換して決定します。
- 出力先は入力 step0001 と同一ディレクトリです。
- 各行は `スタッフコード + step0001 元行` の形式で出力します（先頭に 1 列追加）。

### 補足

- 新ローデータ step0001 は Jobcan 長形式 TSV から生成されます。
- 給与 step0001 は支給・控除 CSV から生成されます。
